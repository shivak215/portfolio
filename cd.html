Abstract-This paper is about forestation survey system. Deforestation is the major cause of soil erosion and landslides which lead to loss of lives. Major reason for this problem is cutting the number of trees more than the authorised number. There is lot of problem faced during tree count in a particular area. In tree counting lot of labour and time is wasted by the traditional methods, to reduce labour and time an application is used to count the number of trees in an area by image processing. This paper will include the solution of the problem of excessive deforestation by motoring particular area by processing the image of that area, which will give the tree count of subjected area, even using this method we can monitor and save the past data to get the statistics and use data analytics on the collected data and work upon it with the time remaining to save the area form any type of natural calamity.

Keywords- Deforestation; Forestation; soil erosion; data analytics; image processing

I.INTRODUCTION

As per the statistics of India State of Forest Report the forest and tree cover is 24.16 percent of the total geographical area [1].A landslide was faced by the people at malin near pune in the year 2014, which was mainly due to the deforestation according to the experts. As per the reports 28000 trees where cut for agricultural purposes but originally the number of trees which were actually cut around 3 lakh [2].

Objective of our work was to give the average number of trees present in a given location using image processing. The work includes processing a satellite image and converting image into red only and taking optimal intensity level, image is converted into a binary image and then algorithm is used to calculate the tree count of the specific region.

To count the tree count algorithm is needed which requires area of the subjected land and satellite view of that land.

II.DIGITAL IMAGE PROCESSING

Digital image is a 2-D matrix and each block of matrix is called a pixel. The value at these pixels is intensity level at that particular point .Modern coloured image is actually an RGB image which is actually a 3 layered 2-D matrix one of red, one of blue, and one of green, these 3 matrix when stacked over one another give a colourful image .

III.WORKING

A picture from satellite is taken in RGB format, blue and green layers are removed from the picture and on the remaining red layer image processing is performed using an algorithm. On this red layer critical intensity level is selected and the image is converted into binary image, all the pixels below

this critical level are converted into 1 and the remaining pixels are set to 0.All the area plotted as 1 is useful area on which algorithm will run.

Tree count is based on the following algorithm According to the data and formulas present the

average crowning area of tree is 4m2 [3].Using

unitary method the number of pixels in a 1m2 area is calculated say the count of pixels be x . Now a loop is executed throughout the matrix and the number of 1s are calculated from the binary image and this count is divided by x to get the tree count.
